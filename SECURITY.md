# Security Policy

## Supported Versions

Currently, OGameX is still in development and has not yet reached a stable version. As such, only the latest version is supported.

| Version | Supported                                      |
|---------|------------------------------------------------|
| < 1.0   | :white_check_mark: (latest release only) |


## AI-Assisted Development

This fork is developed with [Claude Code](https://claude.ai/code) (Anthropic's AI coding assistant). A significant portion of the features, refactors, and bug fixes in this repository were implemented through AI-assisted development sessions. Reviewers and contributors should be aware that code may have been generated or modified by an AI model and should apply the same scrutiny — especially for security-sensitive areas — as they would to any externally contributed code.

## Reporting a Vulnerability

If you discover a security vulnerability within this repository, please send a DM to the maintainer via GitHub. All security vulnerabilities will be promptly addressed.
